# 2D Beach Racing

This is a 2D beach racing game where players can race against each other on a sandy beach. The game includes various features such as multiple levels, power-ups, and multiplayer support.

## Installation

To install the game, clone the repository and run the game using Python.

## Usage

Run the main script to start the game:

```bash
python main.py
```

## Features
- Multiple levels
- Power-ups
- Multiplayer mode
